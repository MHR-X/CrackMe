# CrackMe

تحدي بسيط في هندسة عكسية.

## الهدف
تجاوز التحقق من الاشتراك والوصول لرسالة النجاح.

## طريقة الحل
- حلّل الملف التنفيذي باستخدام x64dbg أو أي أداة هندسة عكسية
- عدّل التعليمات الشرطية اللي تتحقق من `IsPremium()`
- اعرض رسالة "CONGRATS!" للتأكيد على النجاح

## التصنيع (اختياري)
```bash
gcc src/CrackMe.c -o CrackMe.exe -mwindows -s -O2
